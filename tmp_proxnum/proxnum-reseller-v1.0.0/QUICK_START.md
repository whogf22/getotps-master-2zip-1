# 🚀 Proxnum Reseller - Quick Start Guide

Get up and running in **5 minutes**!

## ⚡ Quick Installation

### 1️⃣ Prerequisites (1 minute)

 **You Need:**
- Web hosting with PHP 7.4+ and MySQL
- A domain name
- FTP access or File Manager
- Proxnum account ([Sign up here](https://proxnum.com))

### 2️⃣ Get Your Keys (2 minutes)

**License Key:**
1. Buy license at `/license/purchase` (internal route when running)
2. Get key format: `XXXX-XXXX-XXXX-XXXX`

**API Key:**
1. Login to [proxnum.com](https://proxnum.com)
2. Go to Profile → API Keys
3. Click "Generate New API Key"
4. Copy and save it

### 3️⃣ Upload & Install (2 minutes)

1. **Upload Files:**
   ```
   Download proxnum-reseller.zip
   Upload to your hosting
   Extract in root folder
   ```

2. **Run Installer:**
   ```
   Navigate to: http://yourdomain.com/install
   Follow the wizard (5 steps)
   ```

3. **Delete Installer:**
   ```
   IMPORTANT: Delete /install folder after completion
   ```

## 🎯 First Steps After Installation

### Admin Setup (You)

1. **Login:**
   ```
   URL: http://yourdomain.com/auth/login
   Use admin credentials from installation
   ```

2. **Add Your First Client:**
   ```
   Admin Panel → Clients → Add New Client
   Set name, email, password
   Add initial balance
   ```

3. **Test Purchase:**
   ```
   Login as client
   Buy a number
   Verify it works
   ```

### Client Setup (Your Customers)

1. **Receive Credentials:**
   ```
   Admin creates account
   Client gets email and password
   ```

2. **Login:**
   ```
   URL: http://yourdomain.com/auth/login
   Use provided credentials
   ```

3. **Buy First Number:**
   ```
   Dashboard → Buy Number
   Select service and country
   Receive SMS code
   ```

## 🛠️ Troubleshooting

### "License Error"
-  Check license key is correct
-  Verify license is active
-  Check internet connection

### "Purchase Failed"
-  Check Proxnum account has balance
-  Verify API key is correct
-  Check service availability

### "Page Not Found"
-  Verify .htaccess file exists
-  Check mod_rewrite is enabled
-  Try different browser
